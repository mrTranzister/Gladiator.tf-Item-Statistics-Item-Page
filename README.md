# Gladiator.tf-Item-Statistics-Item-Page
Simplifies item management and analysis
# Installation
1. Install <a href="https://www.tampermonkey.net/" target="_blank" rel="noopener noreferrer">Tampermonkey</a> browser extension
2. Click <a href="https://github.com/mrTranzister/Gladiator.tf-Item-Statistics-Item-Page/raw/refs/heads/main/Gladiator.tf%20%E2%80%93%20Item%20Statistics%20%E2%86%92%20Item%20Page-1.0.user.js" target="_blank" rel="noopener noreferrer"> here to install the script</a> or copy the code manually
3. This script automatically runs on the Gladiator page in the items section.
# Features
Allows you to click on the item name in the item statistics window to open the Items tab additionally, Ctrl+click opens the backpack.tf page.
